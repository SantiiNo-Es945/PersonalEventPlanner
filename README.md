# Personal Event Planner App

## What is this app?
This is an Android app where users can create and manage events, they can add a ttitle, location, category, date and time.

## What can the app do?
- Add new events
- See a list of events
- Edit events
- Delete events

 ## Technologies Used
- Java
- Android Studio

## How does it work?
The app saves the events on the phone, so they are still there when the app is closed.

## How to run the app
Open the project in Android Studio and run it on an emulator or phone.
